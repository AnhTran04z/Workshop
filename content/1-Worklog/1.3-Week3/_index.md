## Week 3 Objectives

- Start learning Unity to prepare for the game project.
- Complete the first AWS labs.
- Get familiar with IAM, VPC, and EC2.
- Understand how to create users, groups, roles, networks, and instances on AWS.

## Tasks to be carried out this week

| Day | Work | Start date | End date | Reference |
| --- | --- | --- | --- | --- |
| Monday-Sunday | Study Unity Engine for the project. Watch Unity tutorial videos and get familiar with Editor, Scene, Game View, Inspector, and Project Window. | 04/05/2026 | 10/05/2026 | Unity tutorial videos |
| Tuesday | Complete Lab 01 - IAM. Create AdminGroup, AdminUser, AdminRole, OperatorUser, and log in as AdminUser. | 05/05/2026 | 05/05/2026 | https://awsstudygroup.com |
| Wednesday | Complete Lab 02 - VPC Networking. Create VPC, configure DNS, create Subnet, Internet Gateway, Route Table, Security Group, Flow Logs, EC2, and NAT Gateway. | 06/05/2026 | 06/05/2026 | https://awsstudygroup.com |
| Thursday | Complete Lab 03 - EC2 Linux/Windows. Create Linux and Windows VPCs, Security Groups, Windows and Linux instances, Snapshot, AMI, custom instance from AMI, attach role, and deploy a Linux application. | 07/05/2026 | 07/05/2026 | https://awsstudygroup.com |
