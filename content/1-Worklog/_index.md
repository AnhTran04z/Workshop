This section presents my internship worklog for the **AWS & Unity RoughLife Project**.  
The worklog is organized over **12 weeks**, covering the process from learning AWS fundamentals, completing AWS labs, preparing the Unity project foundation, developing online co-op gameplay, applying AWS server deployment, and preparing the final internship report.

During the internship, I worked on two main directions:

- **AWS learning and practice:** AWS Free Tier, IAM, VPC, EC2, S3, CloudFormation, CloudWatch, Budgets, AWS CLI, and initial server deployment with GameLift.
- **Unity game project development:** RoughLife, a 2D top-down online co-op boss fight game with lobby, boss room, weapon system, death state, revive mechanic, and dedicated server direction.
