## Event Objectives

The purpose of this event was to create a knowledge-sharing space about AI, cloud operations, enterprise systems, and career development. The event provided practical insights from speakers working in cloud and technology companies, helping students and young engineers understand how AI is being applied in real business environments.

The event also aimed to guide participants in building a sustainable career path in the AI era and understanding how cloud infrastructure can be operated more efficiently with automation and intelligent agents.

## Key Highlights

One of the key topics of the event was career development in the AI era. The speakers shared how students and young engineers can prepare themselves for a future where AI becomes part of daily engineering work. Instead of being afraid of AI, developers should learn how to use AI to improve productivity, automate repetitive tasks, and focus on higher-value problem solving.

Another important topic was Voice AI for the Vietnamese market. The session discussed how Speech-to-Text can be combined with Large Language Models to process Vietnamese conversations more effectively. It also introduced the idea of intelligent handover, where AI can transfer complex cases to humans when needed.

The event also introduced DevOps AI Agent solutions. These agents can help detect incidents, extract logs, build topology diagrams, and suggest mitigation plans. This can reduce system recovery time and help cloud operation teams respond faster when problems occur.

Another useful sharing was about applying AI to HR management. AI can support CV screening, candidate matching, and repetitive HR tasks. However, the speakers also emphasized the importance of data privacy and protecting internal company information.

The event also covered enterprise security architecture, especially private connections and secure system design. This helped participants understand that AI and cloud solutions must be built with security, privacy, and reliability in mind.

## Key Takeaways

- AI can be applied in many areas beyond coding, including cloud operations, HR systems, voice solutions, incident response, and enterprise security.
- Security and privacy are very important when building AI systems for businesses.
- Cloud engineers and developers need to improve continuously in the AI era.
- Technical knowledge, problem-solving ability, and responsible AI usage are important for career development.

## Application to My Project

The knowledge from this event can be applied to the RoughLife project in several ways. For example, the idea of DevOps AI Agent can inspire better monitoring and troubleshooting for game servers. If the game server has errors, logs from CloudWatch can be analyzed more quickly to identify the root cause.

The security knowledge is also useful when designing the AWS architecture for the game. Since RoughLife uses online multiplayer and dedicated servers, the system needs proper network design, secure connections, and clear separation between public and private resources.

The event also helped me understand that deploying a game online is not only about running the server. It also requires monitoring, cost control, security, and reliable operation so that players can have a stable experience.

## Event Experience

This event was a valuable experience because it connected technical topics with real business use cases. I gained a broader view of how AI and cloud are used in enterprise environments, and I also learned how important it is to build secure, maintainable, and practical systems.

Overall, the event helped me become more confident in applying AWS and AI knowledge to my project and future career path.
