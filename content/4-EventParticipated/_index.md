During my AWS/FCAJ learning journey, I participated in three meaningful events. Each event gave me valuable knowledge about AWS Cloud, Artificial Intelligence, DevOps, teamwork, confidence, and real-world project development. These events also helped me improve my technical mindset and gave me more ideas to apply to my final project, **RoughLife**.

Through these events, I learned that building a real technology project is not only about writing code. It also requires cloud knowledge, project planning, communication skills, teamwork, confidence, monitoring, security, and the ability to learn continuously from the community.

### [Event 1](4.1-Event1/)  
&emsp;**Event Name:** FCAJ Community Day - Learning Mindset, AI Prompting, AI-Ready Fresher, and BMAD Method  

&emsp;**Date & Time:** 09:00, May 9, 2026  

&emsp;**Location:** AWS / FCAJ Community Event  

&emsp;**Role:** Participant  


&emsp;**Main Content:** This event focused on learning mindset, AI prompting, AI-ready fresher skills, and the BMAD/BMX method. I learned how to use AI tools more effectively, how to write better prompts, and how to work with AI agents based on clear documentation and structured tasks.  

&emsp;**Value Gained:** I gained a better understanding of self-learning, prompt engineering, and the importance of using AI responsibly. This knowledge can support my RoughLife project when I need to analyze errors, document features, and solve technical problems in Unity, C#, and AWS.
